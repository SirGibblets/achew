# Workflows

Achew offers four main workflows. If you are not sure which to pick, see [Choosing a Workflow](../getting-started/workflows-overview.md).

- **[Smart Detect](smart-detect.md):**  Find chapter cues from the audio itself.
- **[Realign Chapters](realign-chapters.md):**  Fix misaligned chapter timing.
- **[Regenerate Titles](regenerate-titles.md):**  Transcribe new titles at existing timestamps.
- **[Quick Edit](quick-edit.md):**  Load an existing chapter source directly into the editor.

## Completion chime

Some workflows involve long-running processes that can take several minutes. During this time, the progress screen shows a chime control near the bottom (disabled by default): 

![Chime](../img/chime-dark.webp#only-dark){ width="560"; .center}
![Chime](../img/chime-light.webp#only-light){ width="560"; .center}

Clicking the icon will enable an audible chime that plays when processing finishes (if it took more than 15 seconds). This can be useful if you switch to another tab or window while Achew works. Use the row of dots at the bottom to pick from several different chimes.
