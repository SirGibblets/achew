# Regenerate Titles

The regenerate workflow keeps the **timestamps** of a Chapter Reference and transcribes new **titles** from the audio at those timestamps.

![Regenerate](../img/select-workflow-regenerate-light.webp#only-light)
![Regenerate](../img/select-workflow-regenerate-dark.webp#only-dark)

## When to use it

This workflow is for situations where you have a [Chapter Reference](../getting-started/chapter-references.md) that has accurate timestamps, but unsuitable titles.

Some examples:

 - A multi-file book where each file is a chapter, but they're named `Track 01`, `Track 02`, etc.
 - Embedded chapters with generic titles, e.g. `001`, `002`, `003`, and the like.
 - Audnexus chapters that are simply titled `Chapter 1`, `Chapter 2`, and so on.

## Steps

1. [**Select a book**](../getting-started/finding-a-book.md) from Achew's main screen.
2. Choose **Regenerate Titles** on the *Select a Workflow* screen.
3. **Pick a Reference**, e.g. Embedded Chapters. If there are no References, or you wish to use another Reference, click the **Add Chapter Reference** button near the bottom.
4. Click the **Continue with...** button.
5. On the **Transcribe Titles** screen, choose your [transcription settings](../reference/transcription.md). If you have another [Reference](../getting-started/chapter-references.md) with chapters whose timestamps line up with the Reference you originally selected for this workflow, you can optionally enable **Use aligned titles from** to pick and choose titles to use directly without transcribing them. See [Using aligned titles](../reference/transcription.md#using-aligned-titles) for details.
6. Click **Transcribe Titles**. The first few seconds of audio at each chapter timestamp will be transcribed into a title.
7. After transcription has finished, use the [Chapter Editor](../editor/index.md) to review and polish the chapters.
