# Keyboard Shortcuts

## Editor-wide

| Windows / Linux | macOS | Action |
|---|---|---|
| <kbd>Ctrl</kbd>+<kbd>Z</kbd> | <kbd>⌘</kbd>+<kbd>Z</kbd> | Undo |
| <kbd>Ctrl</kbd>+<kbd>Y</kbd> or <br> <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Z</kbd> | <kbd>⌘</kbd>+<kbd>Y</kbd> or <br> <kbd>⌘</kbd>+<kbd>Shift</kbd>+<kbd>Z</kbd> | Redo |

## In a title field
*(If [Tab to Next](overview.md#editor-settings) is enabled)*

| Shortcut | Action |
|---|---|
| <kbd>Tab</kbd> | Jump to next selected chapter's title |
| <kbd>Shift</kbd>+<kbd>Tab</kbd> | Jump to previous selected chapter's title |

## In timestamp/offset fields

| Shortcut | Action |
|---|---|
| <kbd>↑</kbd> / <kbd>↓</kbd> | Nudge timestamp by ±1 second |
| <kbd>Space</kbd> | Start playback from the current value |

Additional shortcuts for inline timestamp editing:

| Shortcut | Action |
|---|---|
| <kbd>Enter</kbd> | Accept and save |
| <kbd>Esc</kbd> | Revert and cancel |

## In any dialog

| Shortcut | Action |
|---|---|
| <kbd>Esc</kbd> | Close the dialog |
